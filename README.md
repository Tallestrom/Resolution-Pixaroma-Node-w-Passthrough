# Resolution-Pixaroma-Node-w-Passthrough
Resolution Pixaroma Node w/ Passthrough for image edit models to passthrough the connected inputs from a get size node out without change when active.
Download the Pixaroma Node pack in comfy manager or github and replace the existing Pixaroma files in ComfyUI\custom_nodes\ComfyUI-Pixaroma\
